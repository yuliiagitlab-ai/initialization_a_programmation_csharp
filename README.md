# Initialization à la programmation C# (Exercises)

This repository contains my C# programming fundamentals practice: small console apps and exercises covering core concepts from variables to arrays, plus a final quiz project.

## Contents

- **01_Variables_Exercices** — variables, types, input/output
- **02_Operateurs_Arithmetiques_Exercices** — arithmetic operators, basic calculations
- **03_Saisie d'un nombre décimal entre Min et Max** — input validation and constraints
- **04_Structures_Iteratives_Boucles_Exercices** — loops (for/while/do..while) and practice exercises
- **05_Tableaux_BoucleFor_Exercices** — arrays and `for` loops
- **06_Sous_Programme_Exercices** — methods / sub-programs
- **07_Projet_finale_quiz_csharp** — final console quiz project
- **08_QuizBelgique_db_text** — quiz variant with text-based data storage

## Tech
